This folder intentionally remains empty in this pass.

I found strong official Rust media candidates, but I did not place them in production because the rights position is nuanced for a commercial/promo-facing site.

Facepunch's fan content guidance permits community websites and screenshots, but it also says written approval is needed for monetising content in ways that use their games to promote someone else's products or services. Because “Real Rust” appears commercially adjacent, I kept Rust-native cards out of the production folder and documented the best candidates in ../review-required/REVIEW_REQUIRED_CANDIDATES.md and docs/MISSING_ASSETS.md.
